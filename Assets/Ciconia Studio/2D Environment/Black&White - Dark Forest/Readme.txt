Readme - B&W Dark Forest
_________________________


The Demo scene included has been setup in Unity 2020.1.0 and using the UniversalRP.
Sprite Shapes Profiles have been set up and placed in Sprites/SpriteShapes. You will need to install the 2D SpriteShape package before unpacking the profiles.
